In this tutorial, we will be using the IDE Android Studio. It is much convenient using an IDE rather than a text editor such as VSCode while you are planning to build a project for an app.

You should also download a JDK (recommended ones are JDK 1.8 and 11). Otherwise, AS will be showing errors. Please follow instructions for installation you can find on the Internet.

The first step is to get Android Studio installed. Installing it from JetBrains Toolbox is highly recommended. It somewhat integrates all JetBrains IDEs. You can download the Toolbox from
https://www.jetbrains.com/toolbox-app/

After you've installed the Toolbox, simply open it and search for "Android Studio" to install. Normally, you will get to see Android Studio by scrolling down in "Tools" for a little.

Then, you need to download the Dart sdk from its official website
https://dart.dev/get-dart

You should also install Flutter from
https://docs.flutter.dev/get-started/install

If you want a simulator of iOS devices, you can download Xcode by type in the following command.
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
It requires macOS. So, it's infeasible to have a running iOS simulator on Windows OSs.

Once you are done, open AS. On the top left, go to File, then click on settings. In Languages & Frameworks, you will see Dart and Flutter. Please tick "Enable Dart support for..." and change the Dark SDK path to where you put the Dark SDK you just downloaded. It looks like this (please only change the settings that are circled, and remain the rest unchanged.


Similarly, do the same to Flutter.


Now go to Plugins. Search for "Flutter" and "Dart", and download both of them.

It's very important that you restart AS after you've done all these settings.

You are all set!
