Visual Flutter, pt.2



Introduction


Just like how we apply css to html, we can apply layouts with constraints to components in Flutter. What's the difference? Well, we have many layouts (flex, inline-box, etc.) in H&C. In Flutter, there are only two layouts you can use:

  • RenderBox
  • RenderSilver

The constrains are just the same as they are in CSS - father components constraint child components. Imagine you have a big div with small divs in it, it's natural that the small ones cannot exceed the size of the big one - their father div.


BoxConstraints


The constructor for a BoxConstraint is: 



      const BoxConstraints({ // this is the constructor of the BoxConstraint class
        this.minWidth = 0.0, // minimum width
        this.maxWidth = double.infinity, // maximum width
        this.minHeight = 0.0, // minimum height
        this.maxHeight = double.infinity // maximum height
      })
