Visual Flutter, pt.1



Introduction


Assuming that you are already familiar with HTML and CSS, writing HTML and CSS code can be really painful since you cannot reuse your code other than copying and pasting. This low efficiency work process can be exhausting. Therefore, people tend to use encapsulated APIs to automatically generate HTML and CSS code, or use different programming languages that are not "Markup" languages, whether they are static or dynamic.

Yes, Flutter provides you with the opportunity to write front-end code and back-end code all together. Let's take a look.


Text, Buttons, Images, Icons, Inputs...

Text

In Flutter, you would create texts by using Text. 


    Text("Hello world",
    textAlign: TextAlign.left,
    );

    Text("Hello world! I'm Jack. "*4,
      maxLines: 1,
      overflow: TextOverflow.ellipsis,
    );

    Text("Hello world",
      textScaleFactor: 1.5,
    );

Note that you can repeat a text by adding *n behind it. This is what it looks like:


TextStyle

you can customize text styles - very similar to CSS:


    Text("Hello world",
    style: TextStyle(
    fontSize: 18.0, // font size of the text
    height: 1.2, // height of a line
    fontFamily: "Courier", // font family of the text
    background: Paint()..color=Colors.yellow, // bgc of the text
    decoration:TextDecoration.underline,
      ),
    );

TextSpan

You can also have a text span to allow you to customize your text blocks: 




    const TextSpan({
      TextStyle style,
      Sting text,
      List<TextSpan> children,
      GestureRecognizer recognizer,
    });

Button

In Flutter, there are many ways of defining a button:


    ElevatedButton( // A button with shadow and gray background
      child: Text("ElevatedButton"),
      onPressed: () {},
    );

    TextButton( // A button without shadow and background
      child: Text("TextButton"),
      onPressed: () {},
    )

    OutlineButton( // An unshadowed button with an outline
      child: Text("OutlineButton"),
      onPressed: () {},
    )

    IconButton( An unshadowed clickable icon with no bg
      icon: Icon(Icons.thumb_up),
      onPressed: () {},
    )


Image

You can load images from both local or web resources


    Image( // load an image from local
      image: asset("img.png"),
    );

    Image(
      image: NetworkImage( // load an image from web resources
          "https://your-image-url"),
      width: 100.0,
    )

You can also customize your image


    const Image({
      ...
      this.width, // width
      this.colorBlendMode, // blend mode
      this.fit,// zoom mode
      this.alignment = Alignment.center, // alignment
      this.repeat = ImageRepeat.noRepeat, // repeat or not
      ...
    })

Icon

Flutter has included Material Design icons. You can enable it in your pubspec.yaml. Use this website to view all icons. 



    flutter:
      uses-material-design: true

You can use icons like this: 


    Row(
      mainAxisAlignment: MainAxisAlignment.center,
      children: <Widget>[
      Icon(Icons.fingerprint,color: Colors.green), // this is an icon
      ],
    )
