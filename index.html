Why choose to use Dart / Flutter instead of the followings?

  • It's a new-gen language - simple syntax; easy to learn
  • Single-threaded. Meaning there are no locks, and no common lags from JVM (especially Android)
  • It supports both JIT and AOT.
  • It is cross-platform. Imagine using one language for both Android and IOS!
  • No more Java-styled stinky, long, annoying code.
